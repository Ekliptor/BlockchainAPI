# BlockCHainAPI
Wrapper project of the SLP-SDK to be used in the browser.

### Installation
```
git clone https://github.com/Ekliptor/BlockchainAPI
yarn install
```


### Compiling
In the project root directory, run:
```
yarn watch
```
